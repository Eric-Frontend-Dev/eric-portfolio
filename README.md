# Eric Portfolio

A modern frontend developer portfolio built with React, Tailwind CSS, Framer Motion and Supabase.

## 🚀 Features

- Responsive modern UI design
- Animated sections with Framer Motion
- Project showcase
- Admin dashboard
- Contact form
- Testimonials section
- Mobile friendly layout

## 🛠️ Technologies Used

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- Supabase
- JavaScript
